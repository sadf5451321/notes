# 📘 AI 大模型应用开发工程师学习计划（16周版）

> 适用对象：会 Python、了解 Web，希望进入 AI 大模型应用开发岗位  
> 学习时间：每天约 5 小时  
> 目标：能独立开发、部署大模型应用（含 RAG、Agent、全栈部署）

---

## 🧩 阶段规划总览

| 阶段 | 周数 | 学习主题 | 学习重点 | 推荐实践项目 | 每天建议安排（5小时） |
|------|------|------------|------------|------------------|------------------|
| 🧩 **阶段1：后端与基础打牢** | 第1-2周 | FastAPI + 异步编程 + ORM | - RESTful API<br>- 异步 async/await<br>- 数据模型与Pydantic<br>- SQLite/PostgreSQL 基础 | 🎯 学生信息管理系统（增删改查） | 3h 实战 + 2h 复盘 |
| 🤖 **阶段2：大模型调用与Prompt设计** | 第3-5周 | OpenAI / 通义千问 / 智谱AI / Claude API | - 聊天接口调用<br>- Prompt模板与上下文设计<br>- Function Calling<br>- JSON输出与解析 | 🎯 智能问答机器人（ChatGPT风格） | 2h 理论 + 3h 编码 |
| 🧠 **阶段3：RAG与知识库构建** | 第6-9周 | 向量数据库 + 检索增强生成（RAG） | - 文本分块与embedding<br>- 向量数据库（Chroma/Qdrant）<br>- 检索召回与重排<br>- RAG架构整合 | 🎯 企业文档知识库问答系统 | 1h 理论 + 4h 实践 |
| ⚙️ **阶段4：Agent与工具调用** | 第10-12周 | LangChain Agent / LangGraph / CrewAI | - 多智能体协作框架<br>- 工具调用(Function Calling)<br>- 自动执行任务Agent | 🎯 AI 助手管家（带联网与计算功能） | 1h 理论 + 4h 项目 |
| 🌐 **阶段5：前端集成与部署** | 第13-16周 | React + Next.js + TailwindCSS + Docker | - 前后端交互接口<br>- 部署FastAPI后端<br>- 容器化与Vercel部署 | 🎯 全栈AI应用平台（登录+对话+知识库） | 2h 前端 + 3h 部署 |
